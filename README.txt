

* The code should be used with STATA software.

* The “Sensspec” Stata module is required to compute sensitivity and specificity. This module should be installed from within Stata by typing "ssc install senspec". 

* Data file should be imported in Stata :
** With 4 variables: AB42, AB40, Tau, pTau (use the code "code with amyloid ratio").
** OR With 3 variables: AB42, Tau, pTau (use the code "code with CSF AB42").
